# NEON Multi-Trophic Biodiversity

## Overview
This project investigates how canopy structural and functional diversity
derived from NEON AOP (LiDAR + hyperspectral) predicts multi-trophic biodiversity.

## Objectives
- Quantify plant, insect, vertebrate, and microbial diversity
- Derive structural (LiDAR) and functional (hyperspectral) metrics
- Link remote sensing to biodiversity across scales

## Data Sources
- NEON field data (DP1)
- NEON AOP data (LiDAR + hyperspectral)

## Project Structure
- `data_raw/`: raw NEON downloads (not tracked)
- `data_processed/`: cleaned datasets
- `scripts/`: data processing and analysis
- `outputs/`: figures and results

## Reproducibility
This project uses `renv` for package management.
